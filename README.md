# Execução do projeto

### Instalação 

Instale o NodeJs pelo link: https://nodejs.org/en/download/

**Clone o projeto**
``` 
git clone https://github.com/julio958214/NPM-Cypress.git
```

**Instale as dependências do projeto**

```
npm install
```

**Execute o projeto**

```
npm run test
```
